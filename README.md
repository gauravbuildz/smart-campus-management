# 💡 The Problem
### In most colleges, students struggle to find centralized information:
* **📱 Important notices are lost in endless WhatsApp groups.**
* **📝 Raising a complaint for a broken hostel fan or Wi-Fi requires physical paperwork.**
* **🔍 Recovering lost items relies entirely on luck.**

<br />

# 🚀 Our Solution
### **Smart Campus Service Hub** is a centralized web application that digitizes daily college operations. It acts as a single digital hub for students and administrators.

---

# ✨ Core Features

### 📌 1. Notice Board
**View real-time, categorized notices and event updates.**

### 🛠️ 2. Complaint System
**Raise issues (Hostel, Wi-Fi, Classroom) instantly and track their status.**

### 🎒 3. Lost & Found
**Report lost items or upload pictures of found items easily.**

---

# 💻 Tech Stack
### • **Frontend:** Next.js 14, React, Tailwind CSS
### • **Backend:** Next.js API Routes 
### • **Database:** PostgreSQL (Neon) with Prisma ORM
### • **Authentication:** NextAuth.js
### • **Deployment:** Vercel

---

# 🛠️ Run Locally 


### **1. Clone the repository**
```bash
git clone [https://github.com/gaurav-spnrec/smart-campus-management.git](https://github.com/gaurav-spnrec/smart-campus-management.git)
cd smart-campus-management
```

### **2. Install dependencies & Run**
```bash
npm install
npx prisma db push
npm run dev
```
